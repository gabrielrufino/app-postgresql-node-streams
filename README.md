# App - PostgreSQL & Node Streams

Streaming data from PostgreSQL on Node.js

### Running project

##### 0) Requirements

* Node.js & npm
* Postgres

##### 1) Clone repository

```
git clone https://github.com/gabrielrufino/app-postgresql-node-streams.git
```

##### 2) Restore dependencies

```
npm ci
```

##### 3) Environment variables

```
cp .env.example .env
```

##### 4) Seed database

```
npm run seed
```

##### 5) Stream database

```
npm start
```
